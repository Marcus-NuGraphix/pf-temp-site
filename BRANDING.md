# Branding Notes

This repository contains the Portuguese Forum temporary holding site.

## Domain and Mail
- Primary domain: `https://portugueseforum.org.za`
- Privacy contact: `info@portugueseforum.org.za`
- Delete requests: `delete@portugueseforum.org.za`
- Mail sender: `no-reply@portugueseforum.org.za`

## Brand Assets
Current production logo assets are PNG:
- `assets/png/logo.png`
- `assets/png/logo-white.png`
- `assets/png/` is now used as the logo source in page markup and `site.config.json`.

## Config Source
`site.config.json` remains the single config source for brand/domain/contact metadata when regenerating output with `npm run brand:apply`.
